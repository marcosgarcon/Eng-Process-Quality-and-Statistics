Guia de Utilização - Cronoanálise MTM Avançada

Novidades da Versão 2.0

  • Base MTM Expandida: Inclui MTM-1, MTM-UAS, MOST e dados ergonômicos
  • Atalhos de Teclado: Navegação rápida e controles de vídeo
  • Análises Estatísticas: Gráficos e métricas de produtividade
  • Templates: Modelos pré-configurados para processos comuns
  • Validação de Qualidade: Verificações automáticas e alertas
  • Controles de Vídeo Avançados: Frame-by-frame, marcadores, velocidade variável

Atalhos de Teclado

  • F1 - Mostrar/ocultar atalhos
  • Espaço - Play/Pause do vídeo
  • Ctrl + N - Nova cronoanálise
  • Ctrl + S - Salvar cronoanálise
  • Ctrl + E - Adicionar etapa
  • Delete - Excluir etapa selecionada
  • ← / → - Navegar frame por frame
  • Shift + ← / → - Pular 5 segundos
  • M - Marcar ponto no vídeo
  • C - Capturar screenshot

Base de Dados MTM

A versão 2.0 inclui uma base completa com mais de 100 códigos MTM pré-configurados:

  • MTM-1: Movimentos básicos (Reach, Move, Turn, Grasp, Position, Release)
  • MTM-UAS: Sistema de análise universal com 77 códigos
  • MOST: Modelos de sequência para atividades complexas
  • Dados Ergonômicos: Avaliação EAWS integrada

Análises Estatísticas

Visualize tendências e padrões em suas análises:

  • Distribuição de tempos por operação
  • Análise de Pareto dos códigos MTM mais utilizados
  • Métricas de eficiência e produtividade
  • Comparação entre analistas e períodos

Validação de Qualidade

O sistema agora inclui verificações automáticas:

  • Detecção de discrepâncias entre tempos observados e padrão
  • Alertas para movimentos com alto risco ergonômico
  • Sugestões de melhoria baseadas em boas práticas
  • Indicadores visuais de qualidade da análise
